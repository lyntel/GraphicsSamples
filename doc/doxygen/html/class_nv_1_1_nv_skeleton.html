Detailed Description

NvSkeleton holds an array of nodes that comprise the hierarchy of transformations for the model.

These nodes form the basis for applying hierarchical transformations to meshes within a model as well as skinning those meshes. An NvSkeleton is a tree where each node has at most one parent, and the index of a parent node MUST be less than that of all of its children. This allows all transforms to be propagated through the tree with a single sequential traversal of the array and any operations performed on nodes in such a traversal to be performed on any child's parent before the child itself. Note that a skeleton may contain more than one "tree" (i.e. multiple nodes with no parent node), but by definition, node 0 is the root of a tree.
